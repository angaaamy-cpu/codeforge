# اتفاقيات المشروع

## قاعدة المحاولات الثلاث

كل مهمة أو مشكلة تتبع دورة المحاولات الثلاث:

### المحاولة الأولى
- المحاولة الأولية للحل
- توثيق المحاولة والنتيجة
- إذا نجحت → إغلاق المهمة

### المحاولة الثانية
- تحليل سبب فشل المحاولة الأولى
- تجربة نهج مختلف
- توثيق الدروس المستفادة

### المحاولة الثالثة
- طلب مساعدة من الفريق
- مشاركة السياق الكامل
- إذا نجحت → إغلاق + توثيق

### بعد 3 محاولات فاشلة
- تصعيد للمشروع
- إعادة تقييم المتطلبات
- طلب تدخل architect

## ADR (Architecture Decision Records)

### متى نكتب ADR؟
- كل قرار معماري مهم
- اختيار تقنية جديدة
- تغيير في تدفق البيانات
- تغيير في هيكل المشروع

### صيغة ADR
```
# ADR-XXX: العنوان

## الحالة
مقبول | مرفوض | متروك

## المشكلة
وصف المشكلة التي نواجهها

## السياق
السياق المحيط بالمشكلة

## خيارات القرار
### 1. الخيار الأول
الوصف + المزايا + العيوب

### 2. الخيار الثاني
الوصف + المزايا + العيوب

## القرار
ما تم اختياره ولماذا

## النتائج
### إيجابية
...

### سلبية
...
```

### موقع الـ ADRs
جميع الـ ADRs في: `docs/adr/`

## Git Conventions

### Branch Naming
```
feature/<description>
bugfix/<issue-number>
hotfix/<critical-issue>
docs/<topic>
```

### Commit After Each Task
كل مهمة تنتهي بـ commit:
```bash
git add .
git commit -m "feat: description of changes"
```

### PR Requirements
- عنوان واضح
- وصف مختصر
- رابط للـ issue
- ملاحظات للمراجعة

---

##Communication

### Channels
- **المستندات**: القرارات المهمة
- **GitHub Issues**: المهام والمتطلبات
- **Pull Requests**: الكود والتغييرات

### Response Times
| النوع | الوقت |
|-------|-------|
| سؤال عاجل | خلال ساعة |
| bug | خلال 4 ساعات |
| feature request | خلال 24 ساعة |
| code review | خلال 48 ساعة |

---

## Code Ownership

### المسؤوليات
- **Architect**: قرارات المعمارية
- **Manager**: تنسيق المشروع
- **Developer**: تنفيذ المهام
- **QA**: جودة الكود

### المراجعة المطلوبة
- تغييرات معمارية → Architect
- تغييرات كبيرة → Manager + Architect
- تغييرات عادية → 1 موافق
- hotfixes → Manager فقط

---

## Definition of Done

مهمة تُعتبر منتهية عندما:
- [ ] الكود مكتوب ومُراجع
- [ ] الاختبارات تمر
- [ ] التوثيق محدث
- [ ] commit في Git
- [ ] مرتبط بـ issue
