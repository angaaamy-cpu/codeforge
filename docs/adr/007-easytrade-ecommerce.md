## المشروع
**EasyTrade** - منصة تجارة إلكترونية بسيطة

---

## السياق

مشروع إنتاجي حقيقي لاختبار منصة CodeForge. الهدف:
- اختبار workflow كامل
- بناء تطبيق متكامل
- تقييم قدرات المنصة

### 3. التصميم

**اللون الأساسي:**
- الخلفية: #0f0f1a (داكن جداً)
- البطاقات: #1a1a2e
- التمييز: #00d4ff (سماوي)
- النص: #ffffff

**RTL Support:**
- dir="rtl" للغة العربية
- dir="ltr" للغة الإنجليزية
- CSS variables للتنسيق

**Responsive:**
- Mobile-first
- Grid layout
- Flexbox

### 4. سلة التسوق

**localStorage:**
```javascript
{
    "cart": [
        {"id": 1, "quantity": 2}
    ]
}
```

**Functions:**
- addToCart(productId)
- removeFromCart(productId)
- updateQuantity(productId, quantity)
- getCartTotal()
- clearCart()

### 5. البحث والتصنيف

**البحث:**
- بحث حر في الاسم والوصف
- تحديث فوري للنتائج

**التصنيف:**
- جميع الفئات
- إلكترونيات
- ملابس
- أحذية
- إكسسوارات

### 6. الصفحات

**index.html:**
- شريط بحث
- أزرار التصنيف
- شبكة المنتجات
- رابط السلة

**product.html:**
- عرض المنتج (ID من URL)
- صورة كبيرة
- الاسم، السعر، الوصف
- زر إضافة للسلة

**cart.html:**
- قائمة المنتجات في السلة
- أزرار +/- للكمية
- المجموع الفرعي والإجمالي
- زر إتمام الشراء

**confirmation.html:**
- رسالة شكر
- رقم الطلب
- زر العودة للرئيسية

**404.html:**
- رسالة "الصفحة غير موجودة"
- زر العودة للرئيسية

### 7. الأمان

**الفحوصات:**
- Input validation لكل المدخلات
- No innerHTML - استخدام textContent
- No eval() أو exec()
- Sanitization للبيانات
