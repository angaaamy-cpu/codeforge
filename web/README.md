# CodeForge Web UI

واجهة الويب لمنصة CodeForge — تجمع بين تجربة Bolt (محادثة + معاينة حية) و Replit (شجرة ملفات + وكلاء + لوحات).

## التشغيل

```bash
cd web
npm install
npm run dev
```

ثم افتح: http://localhost:5173

## الميزات

- واجهة عربية RTL بتصميم احترافي داكن
- محادثة وصف المشروع + معاينة حية
- مؤشرات خطوات مضغوطة (لا فيضان نصي)
- شريط وكلاء حيّ (Manager / Architect / Developer / QA)
- شجرة ملفات + سجل المشاريع
- حفظ المشاريع في Supabase

## التقنيات

- Vite + React + TypeScript
- Tailwind CSS
- Supabase (مشاريع + سجل)
